# Nyota Full-Stack Project

## Structure
- `frontend/`: React app (Create React App)
- `backend/`: Node.js + Express API

## Getting Started

### Frontend
```
cd frontend
npm start
```

### Backend
```
cd backend
node index.js
```

---

## Development
- Keep frontend and backend code separated.
- Use environment variables for secrets.
- See each folder's README for more details (add as needed).
